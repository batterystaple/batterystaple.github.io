                Notes

            
    
                
  • Do not try to find a password which uses words you like or know. Doing so will make your
                    password weaker.
                
    • 
                
  • Adding spaces is not recommended as it does not make the password safer. Also, some systems do not
                    allow spaces in passwords.
                
    • 
                
  • Capitalizing words or including random numbers does not make password guessing significantly harder,
                    but may be necessary to satisfy the password requirements of some systems.
                
    • 
            


        


        

            
Inspired by the "correct horse battery staple" scheme outlined in XKCD
                #936: Password strength. Some background on this can be found on Security Stack Exchange: Short complex password, or long dictionary
                passphrase?


            
Based on a list of the most common {counting...} english words. All words only use the characters a-z and
                default to lower case.


            
Settings are stored in your browser's localStorage area for your next visit.


            
Words are chosen using client-side JavaScript code that does not use
                any libraries. The code uses a cryptographically secure random number generator.
